SWF Files

SWF Files, also referred to as ShockWave Flash files or Small Web Format files are files that contain Flash Player content, such as games or animations. They can be played using the official projectors, or any other alternative player. Previously, they could be played using the Flash Player plugin, but this is now non-functional.
