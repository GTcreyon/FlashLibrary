TAS63

TAS63 is a collective term for a set of tools allowing tool-assisted speedruns to be performed in Super Mario 63.

Players

TAS63Player

TAS63 Player, or t63p for short, is a modification of the original game SWF to implement TAS functionality.

Editors

TAS63Editor

TAS63Editor, or t63e for short, was an input editor built in C#, using Windows Forms. It was deprecated in favor of [tas63editor-gd].

tas63editor-gd

tas63editor-gd, or t63e-gd for short, is an input editor built in the Godot engine.
